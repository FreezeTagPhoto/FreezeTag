## About the Project

**Free**zeTag is a FOSS project meant to provide an extensible solution for your photo storange needs. Upload all of your photos and have them organized by a robust tagging system, enhanced by our robust [plugin](#plugins) system. 

## Plugins

FreezeTag plugins are written in Python, and used to extend the core functionality of the app to include things like auto-tagging, facial recognition, image captioning, and more. FreezeTag will ship with a number of core first party plugins developed by our team.
